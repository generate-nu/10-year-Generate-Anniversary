# Generate Anniversary Project

This folder was organized to make the Generate archive and current data easier to browse for non-technical reviewers.

## Start here
- Open `00_Start_Here/index.html` in a browser for the clean overview.
- Read `00_Start_Here/README.md` for the folder map.
- Use `04_Summaries_And_Charts/` for chart-ready aggregate files.
- Use `03_Archive_Indexes/` if you need a searchable inventory of what is inside the raw archive zips.

## Folder map
- `00_Start_Here/`: non-technical landing page and quick guide.
- `01_Raw_Source_Archives/`: original archive zip bundles as provided.
- `02_Current_CSV_Data/`: copied current CSV exports used for the project.
  Non-CSV source extras were placed into `02_Current_CSV_Data/Other_Non_CSV_Files/` to keep the main folder clean.
- `03_Archive_Indexes/`: csv indexes of archive contents, semesters, and file types.
- `04_Summaries_And_Charts/`: clean aggregate outputs and chart-ready counts.
- `05_Presentation_Assets/`: space for slides, screenshots, final charts, or polished exports.

## Important note
The raw archive bundles may contain historical internal material that is not presentation-ready. For sharing, use the summary files and dashboard first.
