# Generate — 10 Year Anniversary Data Report

Interactive dashboard visualizing a decade of Generate at Northeastern University.

## What's Inside

- `VIEW_ME_FIRST.html` — Open this in any browser. The full animated, interactive report.
- `00_Start_Here/` — Quick-start guide and lightweight overview
- `01_Raw_Source_Archives/` — Original Google Drive archive bundles (7 zips)
- `02_Current_CSV_Data/` — All F25 application and onboarding CSVs
- `03_Archive_Indexes/` — Searchable manifests of every archived file
- `04_Summaries_And_Charts/` — Executive summary, talking points, metrics JSON
- `05_Presentation_Assets/` — Ready-to-use assets for slides/decks
- `06_Selected_Historical_Data_Extracts/` — Key files pulled from archives

## Export

Every chart has its own CSV download button. Use "Download Everything" for a single comprehensive CSV with percentages, ranks, and totals. JSON export available for programmatic use.

## Stack

Pure HTML/CSS/JS — no build step, no dependencies, no server. Just open the HTML file.
